## Description

Natural Herbs Lite is a herbal WordPress theme and template which is responsive and can be used for fresh, organic, nature, eco friendly, solar, green energy, vegetative, farming, agriculture, spices based. Ayurveda, Yoga, Naturopathy, Medical and Medicines, Spa, stress relief and conservation websites can also use it. It can also be used for multipurpose and is Gutenberg and WooCommerce compatible for online shop and store. It comes with a ready to import Elementor template plugin as add on which allows to import 150+ design templates for making use in home and other inner pages.
